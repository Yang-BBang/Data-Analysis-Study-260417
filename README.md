# 📊 DATA-ANALYSIS-STUDY

R과 Python을 활용하여 데이터 분석 역량을 쌓아가는 학습 기록 저장소입니다. 기초 문법부터 시작하여 **당뇨병 예측** 및 **대사체 데이터 분석**과 같은 실전 프로젝트 수행 과정을 담고 있습니다.

## 📂 Repository Structure

### [Python-Language](./Python-Language)
* **Python-Basic**: 파이썬 프로그래밍 기초 및 제어문 학습
* **Python-Data-Analysis**: Pandas, NumPy를 활용한 데이터 전처리 및 시각화 실습

### [R-Language](./R-Language)
* **R-Basic**: R 통계 프로그래밍 기초 문법 학습
* **R-Data-Analysis**: 데이터 기반 실전 분석 및 예측 모델링
  * **당뇨병 유무 예측 및 핵심 인자 발굴**: 환자 데이터를 활용한 분류 모델링 및 주요 변수 중요도 분석
  * **대사체 데이터를 활용한 질환 핵심 마커 발굴**: 고차원 생물학적 데이터를 활용한 통계 분석 및 유의미한 바이오마커 추출

## 🚀 Key Projects

### 1. 당뇨병 예측 및 인자 발굴 (R)
- **분석 목적**: 특정 생체 지표를 기반으로 당뇨병 발생 여부를 예측하는 모델을 구축하고, 발병에 기여하는 핵심 인자를 식별합니다.
- **기법**: 로지스틱 회귀, 랜덤 포레스트 등 머신러닝 알고리즘 활용.

### 2. 대사체 질환 마커 발굴 (R)
- **분석 목적**: 대사체학(Metabolomics) 데이터를 분석하여 정상군과 질환군 사이의 유의미한 차이를 보이는 대사 물질을 식별합니다.
- **기법**: PCA, T-test, Volcano Plot 등을 통한 통계적 유의성 검정.
